# Hub Beyond Numbers | Math Learning Hub

An elegant, premium mathematical assessment platform designed for Basic 5 through Basic 9 students. This application provides a curated question bank with interactive trials, focusing on making mathematics "The Easy Way" through high-end UI/UX and pedagogical excellence.

## ✨ Features

* **Premium Aesthetic:** Designed with an "Obsidian & Gold" theme using the *Cormorant Garamond* and *Inter* font families for a professional, academic feel.

* **Grade-Based Navigation:** Seamless switching between Basic 5, 6, 7, 8, and 9 curriculum standards.

* **Interactive Trials:** Three distinct trial sessions per grade, each containing 60 curated questions.

* **Live Performance Tracking:** Real-time scoring and percentage calculation as students progress through assessments.

* **Mobile Responsive:** Fully optimized for smartphones, tablets, and desktops.

* **Static Deployment:** Ready for GitHub Pages with zero external dependencies.

## 🛠️ Technology Stack

* **HTML5:** Semantic structure.

* **CSS3:** Advanced layouts, glassmorphism effects, and custom animations.

* **JavaScript (ES6):** Dynamic question generation, state management, and scoring logic.

* **Google Fonts:** Premium typography integration (Cormorant Garamond & Inter).

## 🚀 Deployment

This project is optimized for **GitHub Pages**. To host it yourself:

1. Fork this repository.

2. Ensure your main dashboard file is named `index.html`.

3. Go to **Settings > Pages** in your GitHub repo.

4. Select the `main` branch as the source and click **Save**.

## 📖 Question Bank Structure

The assessment content is derived from the following core strands:

* **Number:** Fractions, percentages, and operations.

* **Algebra:** Substitution, expansion, and simplification.

* **Geometry & Measurement:** Perimeter, area, vectors, and angles.

* **Data/Chance:** Probability and statistics.
